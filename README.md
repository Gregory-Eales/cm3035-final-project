# cm3035-final-project
django social media clone 

```
cm3035-final-project               # root directory of project
│
├── user_management/               # handles user creation, login, logout
│
├── profiles/                      # manages user profiles, search, friend requests, friends
│
├── chat/                          # real-time chatting between friends
│
├── status_updates/                # allows users to post status updates to their homepage
│
├── media/                         # handling of media files like images
│
├── api/                           # implementation of django rest framework for API
│
├── core/                          # URL routing, base models, common utilities
│
├── manage.py                      # command-line utility for administrative tasks
│
├── db.sqlite3                     # default SQLite database (if used)
│
└── socialsite/                    # main project configuration (settings, URLs, etc.)
```
